# Git Commit with Label

  ## What & Why
  Create a labeled git commit capturing the recent merged changes (localization fixes, calculator dropdown fix, post-merge setup) with a specific user-defined commit message.

  ## Done looks like
  - A git commit exists with the message: "Changes to clean up symbols, localization, and unit availability in the calculator."
  - All current working-tree changes are included.

  ## Out of scope
  - Any code changes — this is a commit only, no modifications.

  ## Tasks
  1. **Stage and commit** — Run `git add -A` and `git commit -m "Changes to clean up symbols, localization, and unit availability in the calculator."` from the project root.

  ## Relevant files
  - `scripts/post-merge.sh`
